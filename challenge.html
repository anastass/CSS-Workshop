Challenge

  1. Underline the h1 heading.
  2. Remove the large gap between the header element and the following paragraph
  3. Target the .highlight class and Highlight the text. Give it a yellow highlight.
  4. Target the .justify class and justify the text i.e make the text flush with both margins - left and right.
  5. Target the .center class and center the text.
  6. Target the .caps class and make the text appear in capital letters ONLY!
  7. Target the .indent class and indent the text.
  8. Target the .drop-cap class and give it a drop cap effect.
  9. Target every link in the document, and give them a color of darkblue with a bold weight, EXCEPT the link in the paragraph, .no-link. Give that a color of red
  10. When hovered upon, change the color of the links to teal
